# AI Skill: Project Learning

帮助用户系统性地理解和学习代码项目的 AI 技能定义。适用于 Gemini、Claude Code、Codex、opencode 等多平台。

## 核心理念

**从宏观到微观、从已知到未知** — 先建立全局认知，再逐步深入细节。

## 五阶段学习工作流

1. **项目全景扫描** — 扫描项目结构，输出概览报告、架构图和学习路线图
2. **模块深入分析** — 使用「骨架法」：先看接口/抽象 → 再看组装/入口 → 最后看具体实现
3. **代码逐行解析** — 含语法特性专项讲解
4. **执行链路分析** — 用 Mermaid 序列图追踪调用链路
5. **学习笔记与项目文档整理** — 整合会话内容为持久化产出，含自查复核

## 特色功能

- **双轨产出** — 个人学习笔记（主观）+ 项目文档（客观、可分享）
- **三层记忆** — 长期（`_progress.md`）+ 中期（`_session.md` 会话检查点）+ 短期（对话上下文）
- **骨架法** — 先骨架（接口）→ 再组装（入口）→ 后血肉（实现），避免一上来就陷入细节

## 文件结构

```
├── SKILL.md              # 完整技能定义（核心文件）
├── TODO.md               # 改进想法和版本路线图
└── references/           # 模板文件
    ├── notes_template.md         # 个人学习笔记模板
    ├── progress_template.md      # 跨会话学习进度模板
    ├── session_template.md       # 会话检查点模板
    └── project_doc_template.md   # 项目文档模板
```

## 使用方式

将 `SKILL.md` 加载到你使用的 AI 编程助手中，然后对着目标项目说"帮我学习这个项目"即可。
