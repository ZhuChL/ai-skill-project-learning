# 会话检查点模板（`_session.md`）

会话检查点是**单次学习会话内的中期记忆**，在学习过程中增量更新。
- 减少对话上下文的 token 消耗（AI 可参考此文件而非回看完整对话）
- 防止会话意外中断导致学习状态丢失
- 为整理笔记提供半成品素材

**更新时机**：在自然断点（一个模块分析完毕、一次链路追踪结束、一个重要概念讲透）自动更新，不需要用户触发。

**生命周期**：会话开始时创建/恢复，会话结束整理笔记时内容融入笔记和项目文档，然后清空。

```markdown
# 🔖 会话状态：[项目名称]

> 会话开始：YYYY-MM-DD HH:MM
> 最后更新：YYYY-MM-DD HH:MM

## 当前位置
- 当前阶段：[阶段一/二/三/四/五]
- 正在分析：[模块/函数/流程名称]

## 本次已覆盖
- ✅ [已完成的学习内容1]
- ✅ [已完成的学习内容2]
- 🔄 [正在进行的内容]

## 关键发现（待写入笔记/项目文档）
- [发现1]
- [发现2]

## 用户状态
- 已掌握：[用户已理解的概念]
- 有困惑：[用户表现出困惑的地方]
- 提问趋势：[表层/深层]

## 待处理
- [ ] [还没回答的问题]
- [ ] [计划要讲但还没讲的内容]
```
